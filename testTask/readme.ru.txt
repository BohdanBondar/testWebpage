testTask
Форма регистрации и авторизации пользователя, MySQL, JS, PHP 7.
После копирования на сервер директории testTask, форма будет доступна по ссылке <название сайта>/ testTask. Также файлы с директории testTask можно положить куда угодно на сервер, и форма будет доступна по соответствующей ссылке. Т.е. форма не завязана по абсолютному пути.
Включено GZIP сжатие. 
Произведёно сжатие и минификация CSS стилей.

db/ - дамп базы данных.
Структура файлов:
style.css - главный файл стилей формы и страницы пользователя 
js/ - java script файлы 
script.js - содержит javascript управления регистрационной формой и валидацией введенных данных. 
controllers/php/ - скрипты отвечающие за вход, выход, регистрацию.
setting.php – скрипт содаржит переменные для подключения к базе данных. 
login.php- скрипт отвечает за вход пользователей. 
logout.php – скрипт файл отвечающий за выход пользователя из аккаунта.
registration.php – скрипт файл отвечающий за регистрацию пользователей.
view/ - файлы отображения
userpage.php - заголовочный файл, й выводит данные о пользователе.
img/ - файлы изображений.

index.php - главный файл, который выводит на страницу необходиымую информацию.
